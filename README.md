# DOPE-Toolkit
It is a collection of Windows' utilities. This project is developed in Python. 




### Tools available in the DOPE Toolkit Project
* **WebCam**
* **Browser**
* **Calculator**
* **Calender**
* **Clock**
* **CMD**
* **Color Picker**
* **Crypt Data**
* **Currency**
* **Dictionary**
* **Eggeater Game**
* **Word Count**
* **Stopwatch**
* **Mini Music Player**
* **Notes**
* **QR Code Generator**
* **Text To Speech**
* **Feedback Form**


### __To run the "DOPE.py" successfully, please install the following python packages using pip:__
- pillow
- pypng
- pygame
- pyqrcode
- opencv-python
- pypiwin32
- pywin32
